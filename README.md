# Weatherly

Weatherly is a sleek, React-based weather application built with Vite. It allows users to search for real-time weather forecasts across different cities using the OpenWeatherMap API. 

**Features:**
- Get current weather data including temperature, humidity, wind speed, and pressure.
- Saves recent city searches in an easily accessible history using `localStorage`.
- Toggle between Celsius and Fahrenheit units seamlessly.
